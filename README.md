# beseech

A beautiful command-line prompt for node.js

## Installation

### Installing npm (node package manager)
```
  curl http://npmjs.org/install.sh | sh
```

### Installing beseech
```
  [sudo] npm install beseech
```

## Running tests
```
  vows test/*-test.js --spec
```
